Les Incertitudes – Fiche de Cours

1. Les incertitudes

1.1. Définitions

Les incertitudes sont des erreurs aléatoires lors d’une expérience provenant de la précision des appareils ou de l’expérimentateur.

On considère une grandeur physique notée \( X \) :

  • Valeur vraie : Résultat que l’on obtiendrait pour une mesure parfaite.
  • Valeur mesurée : Résultat obtenu par l’expérience.

1.2. Incertitudes et intervalle de confiance

Les incertitudes peuvent être exprimées de différentes manières :

  • Incertitude absolue : \( u(X) \) estimation de l’erreur de mesure.
  • Incertitude relative : \( \frac{u(X)}{X} \) estimation de l’erreur de mesure en pourcentage.
  • Intervalle de confiance : Estimation pour que la valeur vraie appartienne à un intervalle avec un risque d’erreur fixé.

2. Incertitudes de type A

Lors de la répétition de \( N \) mesures, l’incertitude-type est définie par :

\[ u(X) = k \cdot \frac{s}{\sqrt{N}} \]

où \( s \) est l'écart-type des mesures et \( k \) dépend du niveau de confiance :

  • Pour un niveau de confiance de 68 % : \( k = 1 \).
  • Pour un niveau de confiance de 95,5 % : \( k = 2 \).
  • Pour un niveau de confiance de 99,7 % : \( k = 3 \).

3. Incertitudes de type B

Lors d’une seule mesure, l’incertitude-type est définie par :

  • Instrument sans graduation : \( u(X) = \frac{h}{\sqrt{3}} \), où \( h \) est la précision de l’instrument.
  • Instrument avec graduations : \( u(X) = \frac{graduation}{\sqrt{12}} \).

4. Grandeurs composées

Pour une grandeur composée \( C_A = \frac{C_B V_B}{V_A} \), l’incertitude relative est donnée par :

\[ \frac{u(C_A)}{C_A} = \sqrt{ \left( \frac{u(C_B)^2}{C_B} + \frac{u(V_B)^2}{V_B} + \frac{u(V_A)^2}{V_A} \right) } \]

5. Z-score

Pour comparer une valeur mesurée \( X \) avec une valeur de référence \( x_{\text{ref}} \), on utilise le z-score :

\[ Z = \frac{X - x_{\text{ref}}}{u(X)} \]

Interprétation :

  • Si \( Z \leq 2 \), alors \( X \) et \( x_{\text{ref}} \) sont compatibles.
  • Si \( Z > 2 \), alors \( X \) et \( x_{\text{ref}} \) sont incompatibles.

Exercices d'application

Exercice 1 : Incertitude de type A

On effectue 10 mesures d’une grandeur \( X \) et on obtient les valeurs suivantes : 5,2 ; 5,3 ; 5,1 ; 5,4 ; 5,2 ; 5,3 ; 5,1 ; 5,2 ; 5,3 ; 5,4. Calculez l’incertitude-type pour un niveau de confiance de 95,5 %.

Solution :

La moyenne \( \overline{x} = 5,25 \).

L’écart-type \( s = 0,1 \).

Pour \( k = 2 \) et \( N = 10 \) :

\[ u(X) = 2 \cdot \frac{0,1}{\sqrt{10}} = 0,063 \]

Exercice 2 : Incertitude de type B

Un instrument sans graduation a une précision de \( h = 0,5 \, \text{mm} \). Calculez l’incertitude-type.

Solution :

\[ u(X) = \frac{0,5}{\sqrt{3}} = 0,29 \, \text{mm} \]

Exercice 3 : Z-score

Une valeur mesurée est \( X = 10,2 \) avec une incertitude \( u(X) = 0,3 \). La valeur de référence est \( x_{\text{ref}} = 10,0 \). Calculez le z-score et concluez.

Solution :

\[ Z = \frac{10,2 - 10,0}{0,3} = 0,67 \]

Comme \( Z \leq 2 \), les valeurs sont compatibles.
