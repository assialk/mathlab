Nombres Complexes - Fiche de Cours

1. L’idée des nombres complexes

Les nombres complexes ont été introduits pour résoudre des équations polynomiales de degré \( n \geq 1 \), notamment celles qui n'ont pas de solutions réelles.

Exercice d'application

Résoudre l'équation \( x^2 + x + 1 = 0 \) dans \(\mathbb{C}\).

Solution :

Le discriminant est \(\Delta = 1^2 - 4 \times 1 \times 1 = -3\). Les solutions sont donc :

\[ z_1 = \frac{-1 + i\sqrt{3}}{2} \quad \text{et} \quad z_2 = \frac{-1 - i\sqrt{3}}{2} \]

2. Ensemble des nombres complexes

L'ensemble des nombres complexes, noté \(\mathbb{C}\), contient les nombres réels et un nombre imaginaire \(i\) tel que \(i^2 = -1\).

Exercice d'application

Montrer que \(i^3 = -i\).

Solution :

On a \(i^3 = i^2 \times i = (-1) \times i = -i\).

3. Nombre complexe

a. Définition

Un nombre complexe \(z\) s'écrit sous la forme \(z = x + iy\), où \(x\) est la partie réelle et \(y\) est la partie imaginaire.

Exercice d'application

Soit \(z = 3 + 4i\). Déterminer \(\text{Re}(z)\) et \(\text{Im}(z)\).

Solution :

\(\text{Re}(z) = 3\) et \(\text{Im}(z) = 4\).

b. Égalité de nombres complexes

Deux nombres complexes \(z_1\) et \(z_2\) sont égaux si et seulement si leurs parties réelles et imaginaires sont égales.

Exercice d'application

Déterminer \(x\) et \(y\) tels que \(2x + 3i = 4 + yi\).

Solution :

On a \(2x = 4\) et \(3 = y\), donc \(x = 2\) et \(y = 3\).

4. Opérations sur les nombres complexes

a. La somme

La somme de deux nombres complexes \(z = x + iy\) et \(z' = x' + iy'\) est définie par :

\[ z + z' = (x + x') + i(y + y') \]

Exercice d'application

Calculer \((2 + 3i) + (4 - 5i)\).

Solution :

\[ (2 + 3i) + (4 - 5i) = 6 - 2i \]

b. Le produit

Le produit de deux nombres complexes \(z = x + iy\) et \(z' = x' + iy'\) est défini par :

\[ z \cdot z' = (xx' - yy') + i(x'y + xy') \]

Exercice d'application

Calculer \((1 + i)(2 - 3i)\).

Solution :

\[ (1 + i)(2 - 3i) = 2 - 3i + 2i - 3i^2 = 2 - i + 3 = 5 - i \]

c. Inverse d’un nombre complexe

L'inverse d'un nombre complexe \(z = x + iy\) est donné par :

\[ \frac{1}{z} = \frac{\overline{z}}{|z|^2} \]

Exercice d'application

Calculer l'inverse de \(z = 1 + i\).

Solution :

\[ \frac{1}{1 + i} = \frac{1 - i}{(1 + i)(1 - i)} = \frac{1 - i}{1 + 1} = \frac{1 - i}{2} \]

d. Conjugué d’un nombre complexe

Le conjugué d'un nombre complexe \(z = x + iy\) est défini par :

\[ \overline{z} = x - iy \]

Exercice d'application

Calculer le conjugué de \(z = 3 - 4i\).

Solution :

\[ \overline{z} = 3 + 4i \]

5. Equations du second degré

Pour résoudre l'équation \(az^2 + bz + c = 0\) dans \(\mathbb{C}\), on calcule le discriminant \(\Delta = b^2 - 4ac\).

Exercice d'application

Résoudre l'équation \(z^2 - 4z + 5 = 0\).

Solution :

Le discriminant est \(\Delta = (-4)^2 - 4 \times 1 \times 5 = 16 - 20 = -4\). Les solutions sont :

\[ z_1 = \frac{4 + 2i}{2} = 2 + i \quad \text{et} \quad z_2 = \frac{4 - 2i}{2} = 2 - i \]

6. Représentation graphique des nombres complexes

Un nombre complexe \(z = x + iy\) peut être représenté par un point \(M(x, y)\) dans le plan.

Exercice d'application

Représenter les points \(A(1 + i)\), \(B(-2 + 3i)\), et \(C(4 - 2i)\) dans le plan complexe.

Solution :

Le point \(A\) a pour coordonnées \((1, 1)\), \(B\) a pour coordonnées \((-2, 3)\), et \(C\) a pour coordonnées \((4, -2)\).

7. Forme trigonométrique des nombres complexes

Un nombre complexe \(z = x + iy\) peut s'écrire sous la forme \(z = r(\cos\theta + i\sin\theta)\), où \(r = |z|\) et \(\theta = \arg(z)\).

Exercice d'application

Écrire \(z = 1 + i\) sous forme trigonométrique.

Solution :

On a \(r = \sqrt{1^2 + 1^2} = \sqrt{2}\) et \(\theta = \frac{\pi}{4}\). Donc :

\[ z = \sqrt{2}\left(\cos\frac{\pi}{4} + i\sin\frac{\pi}{4}\right) \]

8. Notation exponentielle

Un nombre complexe \(z = r(\cos\theta + i\sin\theta)\) peut s'écrire sous la forme exponentielle :

\[ z = re^{i\theta} \]

Exercice d'application

Écrire \(z = 2\left(\cos\frac{\pi}{3} + i\sin\frac{\pi}{3}\right)\) sous forme exponentielle.

Solution :

\[ z = 2e^{i\frac{\pi}{3}} \]

9. Formules d'Euler et de Moivre

Les formules d'Euler permettent d'exprimer \(\cos\theta\) et \(\sin\theta\) en fonction de \(e^{i\theta}\) :

\[ \cos\theta = \frac{e^{i\theta} + e^{-i\theta}}{2}, \quad \sin\theta = \frac{e^{i\theta} - e^{-i\theta}}{2i} \]

Exercice d'application

Utiliser la formule de Moivre pour calculer \((\cos\theta + i\sin\theta)^3\).

Solution :

\[ (\cos\theta + i\sin\theta)^3 = \cos(3\theta) + i\sin(3\theta) \]
