1. Les bibliothèques

Les bibliothèques en Python contiennent des fonctions prêtes à l'emploi. Voici quelques bibliothèques couramment utilisées :

  • math : fonctions mathématiques de base.
  • cmath : fonctions mathématiques pour les nombres complexes.
  • fractions : fonctions pour les nombres rationnels.
  • random : fonctions de tirages aléatoires et de probabilités.

Pour importer une bibliothèque, on utilise :

from fichier import *

Exercice d'application

Importez la bibliothèque math et utilisez-la pour calculer la racine carrée de 16.

Solution :

from math import sqrt
resultat = sqrt(16)
print(resultat)  # Affiche 4.0

2. Entrées, sorties et variables

Pour lire une entrée utilisateur :

variable = input("Message")

Pour afficher un message :

print("Message", variable)

Exercice d'application

Écrivez un programme qui demande à l'utilisateur son nom et affiche "Bonjour, [nom] !".

Solution :

nom = input("Entrez votre nom : ")
print("Bonjour,", nom, "!")

3. Types de données

a. Typage, création et initialisation d'une variable

En Python, une variable est typée, créée et initialisée en une seule instruction :

variable = valeur

Exercice d'application

Créez une variable age et initialisez-la avec la valeur 25.

Solution :

age = 25

b. Conversion de type (transtypage)

Pour convertir le type d'une variable :

str()  # Chaîne de caractères
float()  # Nombre décimal
int()  # Nombre entier

Exercice d'application

Convertissez la chaîne de caractères "123" en un entier.

Solution :

nombre = int("123")

c. Variables booléennes

Une variable booléenne peut prendre deux valeurs : True ou False.

Exercice d'application

Créez une variable booléenne est_majeur et initialisez-la à True.

Solution :

est_majeur = True

4. Opérations et calculs

a. Opérations algébriques

Les opérations de base en Python :

+  # Addition
-  # Soustraction
*  # Multiplication
/  # Division
%  # Reste de la division
**  # Puissance
//  # Division entière

Exercice d'application

Calculez le reste de la division de 10 par 3.

Solution :

reste = 10 % 3
print(reste)  # Affiche 1

b. Opérations booléennes

Les opérations booléennes en Python :

not  # NON
and  # ET
or  # OU

Exercice d'application

Écrivez une expression booléenne qui vérifie si une variable age est comprise entre 18 et 25.

Solution :

age = 20
est_dans_intervalle = age >= 18 and age <= 25
print(est_dans_intervalle)  # Affiche True

5. Instructions sélectives

Les instructions conditionnelles en Python :

if condition:
    # Instructions si la condition est vraie
else:
    # Instructions si la condition est fausse

Exercice d'application

Écrivez un programme qui vérifie si un nombre est positif ou négatif.

Solution :

nombre = float(input("Entrez un nombre : "))
if nombre >= 0:
    print("Le nombre est positif.")
else:
    print("Le nombre est négatif.")

6. Les listes de valeur

Les listes en Python sont des collections ordonnées et modifiables :

liste = [a, b, c]  # Crée une liste
liste[n]  # Accède à l'élément à l'indice n
liste.append(a)  # Ajoute un élément à la fin
len(liste)  # Retourne la longueur de la liste
min(liste)  # Retourne le minimum
max(liste)  # Retourne le maximum
sum(liste)  # Retourne la somme des éléments
liste.sort()  # Trie la liste

Exercice d'application

Créez une liste contenant les nombres 5, 10, 15, puis ajoutez 20 à la fin de la liste.

Solution :

liste = [5, 10, 15]
liste.append(20)
print(liste)  # Affiche [5, 10, 15, 20]

7. Instructions itératives

Les boucles en Python :

for n in range(nombre):
    # Instructions répétées

while condition:
    # Instructions répétées tant que la condition est vraie

Exercice d'application

Écrivez une boucle for qui affiche les nombres de 1 à 5.

Solution :

for i in range(1, 6):
    print(i)  # Affiche 1, 2, 3, 4, 5

8. Définir une fonction

Pour définir une fonction en Python :

def nom_fonction(paramètre1, paramètre2):
    # Instructions
    return resultat

Exercice d'application

Écrivez une fonction qui prend deux nombres en entrée et retourne leur somme.

Solution :

def somme(a, b):
    return a + b

resultat = somme(3, 5)
print(resultat)  # Affiche 8

9. Utilisation des fichiers

Pour manipuler des fichiers en Python :

fichier = open("fichier.txt", "r")  # Ouverture en lecture
ligne = fichier.readline()  # Lit une ligne
fichier.close()  # Ferme le fichier

Exercice d'application

Écrivez un programme qui lit un fichier texte et affiche son contenu.

Solution :

fichier = open("fichier.txt", "r")
contenu = fichier.read()
print(contenu)
fichier.close()

10. Fonctions mathématiques

La bibliothèque math fournit des fonctions mathématiques avancées :

from math import sqrt, pow, sin, cos, pi

Exercice d'application

Calculez la valeur de \(\sin(\pi/2)\) en utilisant la bibliothèque math.

Solution :

from math import sin, pi
resultat = sin(pi / 2)
print(resultat)  # Affiche 1.0

11. Représentation graphique

La bibliothèque matplotlib permet de créer des graphiques :

import matplotlib.pyplot as plt
plt.plot(x, y)
plt.show()

Exercice d'application

Créez un graphique simple avec les points (1, 2), (2, 4), (3, 6).

Solution :

import matplotlib.pyplot as plt
x = [1, 2, 3]
y = [2, 4, 6]
plt.plot(x, y)
plt.show()
