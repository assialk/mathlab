Sommes de Variables Aléatoires

1. Variables Aléatoires Réelles (VAR)

a. Définition

Soit \(\Omega\) un univers associé à une expérience aléatoire. On appelle variable aléatoire réelle \(X\) une fonction définie de \(\Omega \rightarrow \mathbb{R}\).

Exercice d'application

Soit une expérience aléatoire consistant à lancer un dé à six faces. Définissez une variable aléatoire \(X\) qui représente le résultat du lancer. Quelle est la loi de probabilité de \(X\) ?

Solution :

La variable aléatoire \(X\) peut prendre les valeurs 1, 2, 3, 4, 5, 6 avec une probabilité de \(\frac{1}{6}\) pour chaque valeur. La loi de probabilité de \(X\) est donc :

\[ P(X = k) = \frac{1}{6} \quad \text{pour} \quad k = 1, 2, 3, 4, 5, 6 \]

b. Opérations

Nous étudierons les opérations suivantes avec des variables aléatoires :

  • Multiplication par un réel : \(\lambda X\)
  • Somme de 2 variables aléatoires : \(X + Y\)

Exercice d'application

Soient \(X\) et \(Y\) deux variables aléatoires indépendantes telles que \(E(X) = 3\) et \(E(Y) = 5\). Calculez \(E(2X + 3Y)\).

Solution :

On utilise la linéarité de l'espérance :

\[ E(2X + 3Y) = 2E(X) + 3E(Y) = 2 \times 3 + 3 \times 5 = 6 + 15 = 21 \]

c. Loi de probabilité

On appelle loi de probabilité de la variable aléatoire \(X\) la fonction qui à chaque valeur \(x_i\) associe la probabilité de l’événement \(P(X = x_i)\).

Exercice d'application

Soit \(X\) une variable aléatoire prenant les valeurs 1, 2, 3 avec les probabilités respectives 0.2, 0.5, 0.3. Calculez \(E(X)\) et \(V(X)\).

Solution :

L'espérance \(E(X)\) est donnée par :

\[ E(X) = 1 \times 0.2 + 2 \times 0.5 + 3 \times 0.3 = 0.2 + 1.0 + 0.9 = 2.1 \]

La variance \(V(X)\) est donnée par :

\[ V(X) = E(X^2) - E(X)^2 \]

Calculons \(E(X^2)\) :

\[ E(X^2) = 1^2 \times 0.2 + 2^2 \times 0.5 + 3^2 \times 0.3 = 0.2 + 2.0 + 2.7 = 4.9 \]

Donc :

\[ V(X) = 4.9 - (2.1)^2 = 4.9 - 4.41 = 0.49 \]

2. Espérance Mathématique / Écart Type

a. Espérance Mathématique

\[E(X) = \sum_{k=0}^n p_k \cdot x_k\]

\[E(aX + b) = aE(X) + b\]

\[E(X + Y) = E(X) + E(Y)\]

Exercice d'application

Soit \(X\) une variable aléatoire telle que \(E(X) = 4\). Calculez \(E(3X - 2)\).

Solution :

On utilise la linéarité de l'espérance :

\[ E(3X - 2) = 3E(X) - 2 = 3 \times 4 - 2 = 12 - 2 = 10 \]

b. Variance

\[V(X) = \sum_{k=0}^n p_k (x_k - E(X))^2\]

\[V(X) = E(X^2) - E(X)^2\]

\[V(X + Y) = V(X) + V(Y)\] pour \(X\) et \(Y\) indépendantes.

Exercice d'application

Soient \(X\) et \(Y\) deux variables aléatoires indépendantes telles que \(V(X) = 2\) et \(V(Y) = 3\). Calculez \(V(X + Y)\).

Solution :

Puisque \(X\) et \(Y\) sont indépendantes, on a :

\[ V(X + Y) = V(X) + V(Y) = 2 + 3 = 5 \]

c. Écart Type

\[\sigma(X) = \sqrt{V(X)}\]

\[\sigma(aX) = |a| \sigma(X)\]

Exercice d'application

Soit \(X\) une variable aléatoire telle que \(\sigma(X) = 4\). Calculez \(\sigma(2X)\).

Solution :

On utilise la propriété de l'écart type :

\[ \sigma(2X) = |2| \times \sigma(X) = 2 \times 4 = 8 \]

3. Application à la Loi Binomiale

a. Définition

Deux variables sont identiquement distribuées lorsqu’elles ont la même loi de probabilités.

Exercice d'application

Soit \(X\) une variable aléatoire suivant une loi binomiale de paramètres \(n = 10\) et \(p = 0.5\). Calculez \(E(X)\) et \(V(X)\).

Solution :

Pour une loi binomiale, on a :

\[ E(X) = np = 10 \times 0.5 = 5 \]

\[ V(X) = np(1 - p) = 10 \times 0.5 \times 0.5 = 2.5 \]

b. Espérance Mathématique

Toute variable aléatoire suivant une loi binomiale peut s’écrire comme une somme de variables aléatoires de Bernoulli indépendantes et identiquement distribuées.

c. Propriétés

Si \(X\) suit une loi binomiale de paramètres \(n\) et \(p\) :

\[E(X) = np\]

\[V(X) = np(1-p)\]

\[\sigma(X) = \sqrt{np(1-p)}\]

Exercice d'application

Soit \(X\) une variable aléatoire suivant une loi binomiale de paramètres \(n = 20\) et \(p = 0.3\). Calculez \(E(X)\), \(V(X)\) et \(\sigma(X)\).

Solution :

Pour une loi binomiale, on a :

\[ E(X) = np = 20 \times 0.3 = 6 \]

\[ V(X) = np(1 - p) = 20 \times 0.3 \times 0.7 = 4.2 \]

\[ \sigma(X) = \sqrt{np(1 - p)} = \sqrt{4.2} \approx 2.05 \]

4. Échantillons de n Variables Aléatoires

a. Définition

Un échantillon de taille \(n\) est une liste de résultats obtenus pour \(n\) répétitions identiques et indépendantes.

b. Propriétés

Soit \(X\) une variable aléatoire de moyenne \(E(X)\), de variance \(V(X)\) et d’écart type \(\sigma(X)\).

Soient \(n\) variables aléatoires \(X_1, X_2, \dots, X_n\) distribuées identiquement suivant la même loi de probabilité.

On appelle \(M_n\) la moyenne de l’échantillon définie par :

\[M_n = \frac{X_1 + X_2 + \dots + X_n}{n}\]

On appelle \(E(M_n)\) l'espérance mathématique de la moyenne de l’échantillon définie par :

\[E(M_n) = E(X)\]

On appelle \(V(M_n)\) la variance de la moyenne de l’échantillon définie par :

\[V(M_n) = \frac{V(X)}{n}\]

On appelle \(\sigma(M_n)\) l'écart type de la moyenne de l’échantillon définie par :

\[\sigma(M_n) = \frac{\sigma(X)}{\sqrt{n}}\]

Exercice d'application

Soit \(X\) une variable aléatoire de moyenne \(E(X) = 5\) et de variance \(V(X) = 4\). Calculez \(E(M_{10})\) et \(V(M_{10})\) pour un échantillon de taille 10.

Solution :

Pour un échantillon de taille \(n = 10\), on a :

\[ E(M_{10}) = E(X) = 5 \]

\[ V(M_{10}) = \frac{V(X)}{n} = \frac{4}{10} = 0.4 \]
